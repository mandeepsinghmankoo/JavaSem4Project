Weather App - Deployment Package
================================

Requirements:
- Java 11 or higher installed on your system

How to Run:
1. Double-click "run.bat" to start the Weather App
2. Enter a city name and click Search
3. View real-time weather information

Files Included:
- WeatherApp.jar (Main application)
- json-20240303.jar (JSON library dependency)
- run.bat (Launch script)
- README.txt (This file)

Troubleshooting:
- If the app doesn't start, ensure Java is installed
- Check your internet connection for weather data
- Make sure all files are in the same folder

Developed using OpenWeatherMap API
